How to Yo-Yo

Yo-Yo Handling Process
    Hold out a yoyo by letting it hanging before the floor to check the string length (Cut string if string is too long)
    Tie a loop at the end of the string to indicate the desired length of your yo-yo
    Start winding the string by holding the yo-yo with the left hand and the index finger over the slot to wind the first loop, then remove index finger to wind the rest of the string
    Put the loop on the middle finger (of the dominant hand) and hold the yoyo on hand with the palm up that the string goes from the finger over the top of the yoyo
Throwing the Yo-Yo
    Throw the Yoyo either forward or downward by the dominant hand opened wide and facing the palm outward
    Turn the dominant hand over so that the palm is facing the floor and sharply tug the second the string is fully extended
    Catch the yoyo by the palm of the dominant hand, turn it over, and repeat process
